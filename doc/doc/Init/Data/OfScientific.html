    • ofScientific : Nat → Bool → Nat → α

    For decimal and scientific numbers (e.g., 1.23, 3.12e10). Examples:

    • OfScientific.ofScientific 123 true 2 represents 1.23
    • OfScientific.ofScientific 121 false 100 represents 121e100

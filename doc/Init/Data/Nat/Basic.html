    def Nat.fold {α : Type u} (f : Nat → α → α) (n : Nat) (init : α) :
    α

    Nat.fold evaluates f on the numbers up to n exclusive, in increasing order:

    • Nat.fold f 3 init = init |> f 0 |> f 1 |> f 2
    Equations
    • Nat.fold f 0 x = x
    • Nat.fold f (Nat.succ n) x = f n (Nat.fold f n x)
